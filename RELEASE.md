## 更新内容

### 新增
- **PNG贴图系统**：将程序化绘制的精灵改为PNG图片文件，支持透明背景
- **动画系统**：新增多帧动画播放
  - player: 3帧呼吸浮动效果
  - npc: 2帧点头动作
  - enemy: 3帧攻击姿态变化
  - portal: 4帧颜色循环旋转效果
  - chest: 2帧开/关状态

### 修复
- 优化动画管理器的帧切换逻辑

### 其他
- 新增AnimationManager类管理动画播放

### 运行方式
解压后双击 `run.bat` 即可运行游戏。

### 系统要求
- Python 3.11+
- Windows
- pygame
